## Time Log App

하루의 시간을 세분화하여 기록하고,  
시간이 소모되는 지점을 명확히 파악하는 것을 목표로 한다.

본 앱의 목적은  
**시간을 의식적으로 사용하며, 낭비되는 구간이 없는 하루를 만드는 것**이다.

---

## 목적

- 하루의 시간을 가능한 한 세밀하게 기록한다
- 무의식적으로 소모된 시간을 식별한다
- 사전 계획 없이도 실제 시간 사용 내역을 남긴다

---

## 접근 방식

- 잠금화면에서 즉시 시간 기록 가능
- 버튼 입력 시점이 곧 기록 시각이 된다
- 기존 기록 구간은 자동으로 종료된다
- 기록 항목의 명명은 사후에 처리한다

---

## 기록 범위

- 실제 수행한 모든 행동
- 대기, 휴식, 비활동 시간 포함

---

## 상태

- 개인 사용 목적
- MVP 단계
